# Nova Land

> 개장 전, 노바랜드의 마지막 탐험가가 되어주세요.

Nova Land는 신규 테마파크 오픈을 기념하는 인터랙티브 이벤트 페이지를 컨셉으로 제작한 Playful Portfolio Project입니다.

사용자는 개장 전 시설 점검 탐험가가 되어 각 어트랙션의 문제를 해결하고, 노바랜드의 개장을 완성하게 됩니다.

---

## Project Overview

- 프로젝트명 : Nova Land
- 타입 : Playful Portfolio Project
- 형태 : Interactive Event Website
- 플랫폼 : PC / Mobile
- 장르 : Theme Park Experience
- 컨셉 : 개장 전 테마파크 점검 탐험

---

## Story

노바랜드 개장을 앞두고 주요 시설에 이상이 발생했습니다.

탐험가는 각 시설을 점검하고 문제를 해결하여 노바랜드의 개장을 완성해야 합니다.

모든 시설 점검이 끝나면 숨겨진 시그니처 어트랙션 **Cosmic Voyage**가 공개됩니다.

---

## Features

### Explorer Registration

- 탐험가 이름 등록
- 랜덤 이름 생성
- 진행 상태 저장

### EVE Control System

- 시설 안내
- 미션 설명
- 진행 상황 관리

### Time System

- Day
- Sunset
- Night

시간에 따라 노바랜드의 분위기가 변화합니다.

### Stamp Book

시설을 완료하면 스탬프와 보상을 획득할 수 있습니다.

- 레일 연결 퍼즐

빛의 정원 복구 미션

에너지 코어 복구 미션

- 우주 여행
- 시공간 체험
- 노바랜드 개장
